# qiansai

嵌入式大赛项目仓库。

## 项目简介

本项目用于参加嵌入式相关赛事，主要包含硬件控制、基础驱动与应用逻辑代码。当前仓库用于沉淀比赛过程中的程序实现与调试记录。

## 目录

> 目录会随着开发推进持续更新。

- `src/`：应用层与核心业务逻辑
- `drivers/`：外设驱动代码（如 GPIO、UART、I2C、SPI 等）
- `bsp/`：板级支持包与底层初始化
- `docs/`：项目文档与说明材料
- `tools/`：脚本、调试或辅助工具

## 如何运行

请根据你的芯片平台与工具链进行编译、下载和调试。一个通用流程如下：

1. **准备环境**
   - 安装交叉编译工具链（如 `arm-none-eabi-gcc`）
   - 安装烧录工具（如 ST-Link/J-Link/OpenOCD）
   - 准备串口工具用于日志查看

2. **获取代码**
   ```bash
   git clone https://github.com/ptclll/qiansai.git
   cd qiansai
   ```

3. **编译工程**
   - 若使用 Makefile：
     ```bash
     make
     ```
   - 若使用 IDE（Keil/STM32CubeIDE 等）：打开工程后直接 Build。

4. **下载到开发板**
   - 使用对应烧录工具将生成的固件（如 `.elf` / `.bin` / `.hex`）下载到目标板。

5. **运行与验证**
   - 上电后通过串口查看日志输出
   - 按功能点逐项验证（按键、LED、通信、传感器等）
